QUEM SOMOS

Somos academicos do 2º período de Sistemas de Informação da PUC/PR, campus Curitiba. Este site foi criado para o Projeto da disciplina de Progaração Web, ministrada pelo Professor Eduardo Lino.
No presente site, utilizamos as linguagens HTML, CSS e JavaScript.
O botão de login, ao clicar direciona o usuário para a página de Login ou Cadastro, quando o usuário não estiver cadastrado no Banco de Dados do Site.
Ao clicar no botão "Produtos" o usuário é redirecionado para a página de Compra de Produtos, sendo que ao comprar o produto, este é adicionado ao Carrinho de Compras.
